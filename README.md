# HoJo :)
 
**HoJo - Let's get things done**

HoJo is a minimal todo app built using Flask, SQLite, HTMX, and Tailwind CSS. It helps users manage daily tasks efficiently and instant updates, no need of page reloads.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Instant UI updates using HTMX
- Persistent storage using SQLite
- Clean design with Tailwind CSS

## Tech Stack
- **Backend:** Flask (Python)
- **Frontend:** HTML, Tailwind CSS
- **Interactivity:** HTMX
- **Database:** SQLite
- **Version Control:** Git & GitHub

## How to Run locally
```bash
git clone https://github.com/your-username/hojo.git
cd hojo
pip install -r requirements.txt
python app.py
```
